DeepSeek 适配版——为 [DeepSeek](https://chat.deepseek.com) 提供时间轴导航与文件夹管理的对话增强工具。

本项目是 [Azurboy/deepseek-voyager](https://github.com/Azurboy/deepseek-voyager) 的维护与更新分支，针对 DeepSeek 平台进行了适配，并同步移植了原项目 [Gemini Voyager](https://github.com/Nagi-ovo/gemini-voyager) 的最新功能与优化。

感谢原作者 @[Nagi-ovo](https://github.com/Nagi-ovo) 和 @[Azurboy](https://github.com/Azurboy) 的优秀工作。

---

## 功能概览

### 时间轴导航（已完成）

- 点击节点快速定位到对应消息
- 悬停预览消息内容
- 标记重要消息（星标/收藏，支持在 Popup 面板中查看并跨页面定位跳转）
- 支持一键启用/关闭自定义时间轴（由于官方已适配原生时间轴，默认关闭以防冲突）
- 开启后支持拖拽调整时间轴位置

### 文件夹管理（已完成）

- 两层级结构：文件夹与子文件夹
- 支持拖拽对话到文件夹，支持折叠与隐藏整个文件夹面板
- 允许在 Popup 中微调文件夹图标大小、字体大小以及左右位移偏移量
- 右键菜单：重命名、复制、删除
- 固定常用文件夹
- 导入/导出文件夹数据，支持跨设备同步

## 安装使用

### 下载最新release包 / 源码包

目前主要适配 Chromium 浏览器（Chrome, Edge 等）。

### 加载到浏览器

- 下载项目 zip 压缩包并解压。
- 打开浏览器扩展管理页面（如 `chrome://extensions`）。
- 开启“开发者模式”。
- 点击“加载已解压的扩展程序”，选择解压后的 `dist_chrome` 目录。
